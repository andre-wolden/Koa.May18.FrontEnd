## Quick start

1. Clone this repo using `git clone https://github.com/koasol/Koa.May18.FrontEnd.git`
2. Navigate to folder.<br />
3. Run `npm install`
4. Run `yarn start`

Open browser and navigate to `http://localhost:2425`

Now you're ready to rumble!
